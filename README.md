# Фибоначчи на Python
вычислитель чисел в фибоначчи

## Функционал
- Складывает последние 4 числа в списке
- Засекает время,за которое происходит расчет
- Называет рандомное число из списка и его номер в нем
## Как запустить
1.Скачай файл fibonacci3.py.
2.Открой терминал в папке с файлом.
3.Выполни команду: python fibonacci3.py
## Используемые технологии
- Python 3.14
